# 安全策略

当前维护版本为 `0.2.x` 和 `main`。请通过 GitHub 的私密漏洞报告功能联系维护者；如果该入口暂不可见，请直接联系仓库所有者，不要公开漏洞细节。

报告应包含受影响版本、复现步骤、影响和建议修复，但不要附带真实 API 密钥、认证数据库或玩家数据。维护者会先确认收到，再评估修复与披露时间。

本仓库不应包含部署秘密。Luanti 世界、玩家数据、SQLite 文件、日志、模型权重和 API 密钥必须保留在忽略提交的部署目录中。

ESIP 0.1 内存路由器只用于本机开发。HTTP sidecar 也只允许绑定 `127.0.0.1`/`::1`，必须使用至少 32 字符的高熵 Bearer 令牌；它不是公网网关。任何跨主机部署都必须增加 TLS、独立身份、持久幂等、速率限制、审计和逐项授权。协议消息不得包含任意代码、原始玩家数据库、令牌或真实身份信息。

`runtime/minetest.conf` 可能包含本机 bridge 令牌，因此整个 `runtime/` 必须保持 Git 忽略。泄露令牌后应立即停止 sidecar、生成新令牌并重新运行 `runtime:prepare`。
